根据《义务教育数学课程标准》，小学数学知识点主要分为以下 **六大维度**：

## 知识点部分

### 1. 数与代数
核心内容，占比最大。包括：
- **数的认识**：自然数、整数、小数、分数、百分数
- **数的运算**：加减乘除四则运算，混合运算，估算
- **数的关系**：倍数、因数、质数、合数、公倍数、公因数
- **初步代数思想**：用字母表示数、简单方程（高年级）


### 2. 图形与几何
- **平面图形**：点、线、角、三角形、四边形、圆
- **立体图形**：长方体、正方体、圆柱、圆锥
- **图形的变换**：平移、旋转、对称
- **位置与方向**：坐标、方位、路线


### 3. 量与计量
- **长度**：毫米→千米单位换算
- **面积、体积、容积**
- **质量**：克、千克、吨
- **时间**：时、分、秒、年月日
- **货币**：元角分


### 4. 统计与概率
- **数据收集与整理**：调查、分类、统计表
- **统计图**：条形图、折线图、扇形图
- **平均数、中位数、众数**
- **可能性**：事件的确定性与不确定性

------------

## 思维方法

### 5. 综合与实践
- 跨学科应用，解决实际问题
- 数学活动、项目式学习
- 培养建模意识和应用能力

---

### 6. 数学思想方法（贯穿始终）
- **数感、符号意识、空间观念**
- **推理能力**（归纳、演绎）
- **模型思想**（用数学描述现实）
- **数据分析观念**
- **应用意识与创新意识**
